# webui example

This directory is a self-contained example repository for **webui**.
Start the server from this directory:

```
cd example
webui serve
```

## What's included

| Section | Type | Description |
|---------|------|-------------|
| Help | content | Markdown documentation files |
| Contacts | data | Address book (CSV + schema) |
| Projects | data | Project tracker (CSV + schema) |
| Inventory | data | Stock inventory (CSV + schema) |

See the **Help** section in the navigation for full usage documentation.
